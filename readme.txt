== Description ==

Manage redirects - Create and manage 301 redirects.  One by one or in bulk.
A simple WordPress plugin that allow you to easily set and manage large number of 301 redirect.


== Installation ==

This section describes how to install the plugin and get it working.

= Installing the Plugin =

*(using the Wordpress Admin Console)*

1. From your dashboard, click on "Plugins" in the left sidebar
1. Add a new plugin
1. Search for "Redirect W"
1. Install "Redirect W"
1. Once Installed, if you want to manually change your domain, go to Settings > Redirect
1. Manage your redirections
